# Hangman Game (Python)

A simple **Hangman game** built in Python!
The player guesses letters of a random animal until they figure out the word.  

---

## Features

- Randomly selects a word from a list of animal names.
- Displays the word progress with underscores for unguessed letters.
- Keeps track of letters that have already been guessed.
- Prints a message if the letter has already been guessed.
- Announces a win when the word is fully guessed.
